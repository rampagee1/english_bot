# 🇬🇧 Telegram-бот для изучения английского языка

Интерактивный Telegram-бот, который помогает пользователям эффективно учить английские слова с помощью умных викторин (квизов). Отличный проект для демонстрации навыков работы с асинхронным Python и встроенной памятью бота.

## ✨ Основной функционал
* **Умное тестирование:** Викторины строго ограничены 10 вопросами для удержания фокуса пользователя.
* **FSM (Конечные автоматы):** Бот использует встроенный механизм состояний для подсчета правильных ответов и текущего шага каждого игрока.
* **Финальная статистика:** По окончании теста пользователь сразу видит свой точный результат.
* **Красивое форматирование:** Все изучаемые слова и варианты ответов автоматически форматируются с заглавной буквы для лучшего визуального восприятия.

## 🛠 Технологический стек
* **Язык:** Python 3.14+
* **Фреймворк:** Aiogram 3.x (Асинхронная библиотека для Telegram API)
* **Архитектура:** Модульная структура кода (разделение на handlers, config, database)
* **Безопасность:** Вынос секретных данных в переменные окружения (`.env`)

## 📂 Структура проекта
* `main.py` — Точка входа, запуск и инициализация бота
* `handlers.py` — Логика обработки команд, квизов и состояний FSM
* `config.py` — Загрузка конфигурации и токенов
* `database.py` — Логика работы с данными
* `.env.example` — Шаблон настроек для быстрого развертывания

## 🚀 Как запустить проект локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd english_bot
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` на основе `.env.example` и вставьте туда ваш токен от BotFather:
   ```env
   BOT_TOKEN=ваш_токен_здесь
   ```

4. Запустите бота:
   ```bash
   python3 main.py
   ```
